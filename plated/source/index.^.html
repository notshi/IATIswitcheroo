IATI Switcheroo
converts your IATI data into csv, json, xml and back again.
CSV ➡ JSON ➡ XML ➡ HTML ➡
Data conversion is made locally in the browser and never sent to an external server.
IATI Switcheroo is a web based version of the dflat command line tool.
Project page, issues and feedback on Github .
