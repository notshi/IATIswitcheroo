IATI Switcheroo converts your IATI data into csv, json, xml and back again.
CSV ➡ JSON ➡ XML ➡ HTML ➡
Data conversion is made locally in the browser and never sent to an external server.
IATI Switcheroo (v0.2) is a web based version of the dflat command line tool.
